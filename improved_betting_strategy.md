## 📋 改进后的投注分类指南

### 分类1：高平率比赛（平率>25%，22场）

**特征**：两队实力接近，互相制约，容易形成平局

**包含比赛**：
1. Switzerland vs Canada (34.0%)
2. Ecuador vs Germany (31.5%)
3. Iran vs New Zealand (31.4%)
4. Switzerland vs Bosnia & Herzegovina (29.9%)
5. Ivory Coast vs Ecuador (29.4%)
... 等22场

**投注建议**：
```
分配策略：设预算为P元

① 平局投注：0.35×P
   - 平局赔率通常3.0-4.0
   - 高概率确保收益稳定

② 主队胜：0.35×P
   - 虽然实力接近，但主场优势
   - 赔率相对较低但稳妥

③ 客队胜：0.30×P
   - 在没有明显优势的情况下
   - 赔率较高

示例（300元分配）：
- 平局：105元（概率30%，赔率3.5）= 368元
- 主胜：105元（概率40%，赔率1.8）= 189元
- 客胜：90元（概率30%，赔率2.8）= 252元
```

### 分类2：中等平率比赛（15%-25%，30场）

```
③ 劣势方胜：0.15×P

示例（300元分配）：
```

## ⚠️ 投注风险控制

### 黑天鹅风险防控

1. **伤病风险**
   - 关注主力球员伤病情况
   - 如有伤兵，降低对该队的投注权重
   - 特别关注门神和前锋伤病

2. **天气因素**
   - 美国六月天气炎热，可能影响对技术型球队的预测
   - 雷雨天气可能导致低分比赛

3. **心理因素**
   - 小组赛首轮：双方都稍保守，易平
   - 小组赛末轮：已出线球队可能放水
   - 已淘汰球队可能激起斗志

4. **裁判因素**
   - 某些裁判牌子严格，导致更多平局
   - 留意历史数据中的异常比赛

### 资金管理原则

```
总资金管理：
- 永远不投超过总资金的10%在单场
- 对于高风险比赛（平率>30%）不投超过5%
- 保留20%的资金作为后续调整

止损规则：
- 连续亏损3场后，停止24小时重新评估
- 单日亏损超过10%，立即停止投注

获利规则：
- 获利达到20%后，提取一半
- 获利达到50%后，提取本金，剩余是纯利

位置管理：
- 不建议all-in任何单项
- 分散投注降低风险
```


## 📌 快速决策流程

对于任何一场比赛：

```
第1步：查看平率预测
  平率>25%？ → 平局应该是主要选项
  平率15-25%？ → 平局作为重要配置
  平率<15%？ → 可以忽视平局

第2步：查看概率大小
  主客概率接近？ → 平率会更高（验证一致性）
  主客概率差异大？ → 倾向主客胜

第3步：查看赔率反馈
  模型 vs 赔率差异大？ → 信赖赔率
  模型 vs 赔率一致？ → 信赖模型

第4步：决定投注分配
  根据平率水平，选择保守/中等/激进配置

第5步：确认风险
  是否超过单笔5-10%限制？
  总敞口是否超过日限？
```
